QR · UTILITY

OFFLINE ANDROID SCANNER + GENERATOR

A small QR-code scanner and generator for Android.

Read codes with the camera or from a saved image, generate a code from any text or URL, and keep a local history log. Optionally connect a database — on-device, LAN, or remote — to batch-generate codes from a table and record scans. No accounts, no tracking.

SCAN

Point the camera at any QR code, or decode one from a photo in your gallery. Torch toggle for low light.

CREATE

Turn any text or URL into a QR code with selectable error-correction (L/M/Q/H) and save it as a PNG.

HISTORY

Every scan and generated code is kept in a local log (last 200), stored on-device only — tap to copy.

DATA

Optional database mode: on-device SQLite, or a LAN/remote PostgreSQL or MySQL. Batch-generate QR PNGs from a query and record scans back.

PRIVATE

No accounts, no analytics, no tracking. Database sync is opt-in and connects only where you point it; credentials are stored encrypted on-device.

Install on your phone

  1. Tap Download APK above (or open this page on the phone and download it there).
  2. Open the downloaded qr-utility.apk.
  3. If prompted, allow your browser / file manager to install unknown apps (one-time).
  4. Open QR Utility and grant the camera permission on first scan.

This is a debug build intended for personal sideloading, not the Play Store.
